# 用户旅程与机会地图

## 一、管理员核心旅程

### 旅程 1：发布虚拟桌面资源

| 阶段 | 动作 | 接触模块 | 机会点 |
| :--- | :--- | :--- | :--- |
| 准备 | 准备虚拟机模板并导入 VDC | 资产 > 虚拟机模板 | 模板兼容性自动检测 |
| 创建 | 新建虚拟桌面资源，选择工作模式、模板、桌面类型 | 服务 > 资源 | 向导式资源创建与依赖提示 |
| 授权 | 创建角色并关联用户/资源 | 服务 > 角色、用户 | 批量授权与策略模板 |
| 验证 | 登录客户端查看资源是否可用 | 运维中心 > 用户状态 | 资源发布状态可视化 |

### 旅程 2：配置安全策略

| 阶段 | 动作 | 接触模块 | 机会点 |
| :--- | :--- | :--- | :--- |
| 定义对象 | 维护应用库、IP 组、时间计划 | 策略 > 对象 | 对象复用与冲突检测 |
| 配置策略集 | 选择对象并配置账号/桌面/应用/物理机策略 | 策略 > 策略集 | 策略影响范围预览 |
| 接入检查 | 定义接入安全规则和策略 | 安全增强 > 接入安全 | 规则预定义库与组合规则 |
| 动态调整 | 按登录 IP/客户端/时间配置情景策略 | 安全增强 > 情景策略 | 策略命中模拟与日志回查 |

### 旅程 3：日常运维与故障排查

| 阶段 | 动作 | 接触模块 | 机会点 |
| :--- | :--- | :--- | :--- |
| 监控 | 查看概览、HCI 状态、在线用户、远程应用状态 | 运维中心 | 统一告警与趋势看板 |
| 处置 | 结束会话、注销用户、解锁用户 | 运维中心 > 用户状态/远程应用状态 | 批量处置与二次确认 |
| 审计 | 查看操作告警、审计、安全、系统日志 | 运维中心 > 日志 | 日志关联分析与导出 |
| 排障 | 使用 Web 控制台命令、抓包工具 | 运维中心 > 故障排查 | 诊断建议与知识库联动 |

## 二、机会地图

| 触点 | 当前痛点 | 机会 | 优先级 |
| :--- | :--- | :--- | :--- |
| 新建资源向导 | 字段多、依赖关系复杂 | 步骤化向导、依赖检测、默认值推荐 | P1 |
| 策略集配置 | 策略域多、对象关系复杂 | 策略影响范围可视化、冲突检测 | P1 |
| 认证配置 | 14 个认证子页，组合多 | 认证方案模板、一键检测 | P2 |
| 日志与告警 | 8 类日志，告警项多 | 统一检索、智能关联、告警降噪 | P2 |
| 权限管理 | 角色权限矩阵未验证 | 按角色预览菜单与操作 | P2 |

## 三、关键触点清单

| 触点 | 模块 | 重要性 |
| :--- | :--- | :--- |
| 服务 > 资源 > 新建虚拟桌面 | 服务 | 高 |
| 服务 > 用户/角色 | 服务 | 高 |
| 服务 > 认证 | 服务 | 高 |
| 策略 > 策略集 | 策略 | 高 |
| 安全增强 > 情景策略/接入安全 | 安全增强 | 高 |
| 资产 > 虚拟机模板/虚拟机 | 资产 | 高 |
| 运维中心 > 概览/用户状态/日志 | 运维中心 | 高 |

## 四、待验证项

- `[OPEN]` 普通管理员/受限管理员的旅程是否与 Super Admin 存在显著差异。
