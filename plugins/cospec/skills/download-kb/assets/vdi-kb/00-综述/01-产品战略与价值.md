# 产品战略与价值

## 一、产品定位

`【推导·待审】` 深信服虚拟桌面控制器（Virtual Desktop Controller, VDC）是 aDesk VDI 解决方案的核心管理组件，面向企业 IT 管理员提供虚拟桌面、远程应用、物理工作站、文件服务、客户端、安全策略和运维中心的统一管理能力。

## 二、目标场景

| 场景 | 说明 | 来源 |
| :--- | :--- | :--- |
| 企业办公桌面云化 | 将 Windows/Linux 桌面以虚拟桌面形式交付给终端用户 | `【推导·待审】` |
| 远程应用与会话桌面发布 | 通过 SBC 1.0/2.0 发布 Windows Server 远程应用和会话桌面 | `【事实·界面】` |
| 3D 设计与托管工作站 | 纳管配备 NVIDIA 独立显卡的物理工作站作为 3D 资源 | `【事实·界面】` |
| 教育/行业应用 | 课程管理等行业应用（当前环境未启用） | `【事实·界面】` |
| 统一接入安全 | 通过策略集、情景策略、接入安全、认证等模块控制访问 | `【事实·界面】` |
| 集中运维与可观测 | 通过运维中心监控服务、许可、告警、日志、用户状态和故障排查 | `【事实·界面】` |

## 三、核心价值主张

| 价值维度 | 说明 | 来源 |
| :--- | :--- | :--- |
| 统一管理 | 资产、服务、策略、安全、运维五大域在一个控制台内完成 | `【事实·界面】` |
| 多终端接入 | 支持 Windows、Mac、Linux、瘦客户机、移动端、Web VDI 等多种客户端 | `【事实·界面】` |
| 灵活资源模型 | 虚拟桌面、远程应用、会话桌面、物理机、文件服务等多种资源类型 | `【事实·界面】` |
| 安全与合规 | 认证、策略集、情景策略、接入安全、外发审计、分布式防火墙等安全能力 | `【事实·界面】` |
| 可观测与运维 | 概览、状态监控、日志、告警、审批、故障排查、抓包等 | `【事实·界面】` |
