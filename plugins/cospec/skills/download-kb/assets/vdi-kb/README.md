# VDI 产品规划知识库

> 本知识库按 `product-planning-kb` 模板结构，对 `VDC界面功能梳理` 五个模块（资产、服务、策略、安全增强、运维中心）的界面功能进行提取与重构。提取不到的信息保留 `待确认` 或放空，不编造。

## 使用说明

1. 本文档覆盖 VDI 5.9.8 EN 控制台在 `admin / Super Admin` 角色下可见的功能。
2. 证据来源分为 `【原文·界面】`/`【事实·界面】`/`【原文·手册】`/`【原文·功能清单】`/`【推导·待审】`/`【待验证】`，与原始梳理文档保持一致。
3. 功能规划文档统一使用 `03-功能规划/` 下的模板格式，每个 VDC 大模块对应一份功能规划文档。
4. 未经验证或无法从界面直接提取的内容标注为 `[OPEN]` 或 `待确认`。

## 标签约定

- `[ASSUMPTION]`：尚未验证的关键假设
- `[EVIDENCE]`：有界面、手册或功能清单支持
- `[DECISION]`：已形成的产品决策
- `[OPEN]`：待确认问题
- `[MVP]`：最小可验证范围
- `[NEW]`：新能力
- `[ENHANCED]`：已有能力增强
- `[OUT]`：明确不在当前范围
- `[BLOCKED]`：被依赖或决策阻塞

## 核心约束

- 本次梳理仅进入页面、展开菜单和打开空白表单，未执行保存、删除、启停、上传、下载等变更操作。
- 部分功能因当前环境未启用或未配置（如分布式防火墙、行业应用、文件库、软件分发容器等），无法下钻完整字段，已在对应文档中标注 `待验证`。
- 当前结论基于 VDI 5.9.8 EN 控制台，5.9.6 中文界面需重新采集核对。
