# 版本范围与路线图

## 一、当前版本范围

`【事实·界面】` 本次梳理覆盖 VDI 5.9.8 EN 控制台以下五大模块：

| 模块 | 覆盖范围 | 验证深度 |
| :--- | :--- | :--- |
| 资产 | 虚拟机模板、虚拟机、物理机、浮动池私有磁盘、远程应用服务器、客户端、文件服务器、打印机 | 页面已读取、交互已展开、未执行变更操作 |
| 服务 | 资源、用户、角色、认证、行业应用、文件、软件分发 | 页面已读取、交互已展开、未执行变更操作 |
| 策略 | 策略集、主题、对象、分布式防火墙 | 页面已读取，分布式防火墙因未启用未下钻规则表单 |
| 安全增强 | 情景策略、临时权限、接入安全、应用特征、Windows 自动更新 | 页面已读取、交互已展开、未执行变更操作 |
| 运维中心 | 概览、HCI 状态、远程应用状态、用户状态、审批、日志、故障排查、重启/关机、关于 | 页面已读取、交互已展开、未执行变更操作 |

## 二、明确在当前范围外/未启用

| 能力 | 状态 | 说明 |
| :--- | :--- | :--- |
| 分布式防火墙规则表单 | `[OUT]` | 当前环境未启用，规则创建、服务对象、高级搜索等无法验证 |
| 行业应用课程管理 | `[OUT]` | 当前环境教育组件未启用，启用后页面未验证 |
| 文件库存储与文件操作 | `[OUT]` | 当前环境未配置文件存储服务器，上传/目录/权限未验证 |
| 软件分发应用容器测试步骤 | `[OUT]` | 当前无应用容器，测试步骤、版本详情、任务结果未验证 |
| 普通/受限管理员视角 | `[OUT]` | 当前仅使用 Super Admin 账号 |

## 三、路线图

| 阶段 | 目标 | 关键任务 | 时间 |
| :--- | :--- | :--- | :--- |
| Phase 1：基线 | 完成界面功能入口梳理 | 输出 VDC 界面功能详解文档（已完成） | 2026-07 |
| Phase 2：验证 | 在测试环境执行变更操作 | 保存/删除/启停/上传/下载等操作结果验证 | 待确认 |
| Phase 3：权限 | 验证不同管理员角色 | 普通/受限管理员菜单与操作权限差异 | 待确认 |
| Phase 4：下钻 | 补齐未启用模块 | 分布式防火墙、行业应用、文件库、软件分发 | 待确认 |
| Phase 5：规划转化 | 将功能梳理转化为产品规划 | 输出 PRD、原型、测试用例、上线 checklist | 待确认 |

## 四、范围变更风险

- `[ASSUMPTION]` 若后续版本 UI 或字段调整，当前梳理文档需要同步更新。
- `[BLOCKED]` 未启用模块的下钻验证依赖测试环境配置，当前无法推进。
- `[OPEN]` 是否需要将 5.9.6 中文控制台作为单独范围进行核对。
