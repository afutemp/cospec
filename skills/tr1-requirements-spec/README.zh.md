# TR1 用户需求说明书生成器 — 中文参考说明

> 本文件仅供人阅读，不加入 AI 上下文。AI 使用的是同目录下的 `SKILL.md`（英文）。

## 概述

将 DCP2-1 立项材料、竞品分析、客户输入、Demo、共创验证、用户旅程和技术评估结论，转换为新版 TR1 用户需求说明书。

## 输出分版

| 版本 | 用途 | 适用场景 |
|------|------|---------|
| 大需求评审版 | 给人评审，讲清楚为什么做、做什么、价值是什么 | 多模块、跨团队需求 |
| 大需求 AI 上下文版 | 给 AI 生成 EPIC/Feature/Story/Tech/测试/编码任务 | 大需求的下游拆解 |
| 小需求评审版 | 单点功能、局部优化快速评审 | 小范围改动 |

## 核心机制

- 双重裁剪：先按任务目的裁剪，再按需求范围裁剪
- 阻塞式材料确认：生成前必须做用户可见的执行前提示
- 来源 ID 保留：FEAT/BR/AC/DEMO/NFR 等 ID 必须在输出中保留

## 输入来源

DCP2-1 材料、竞品分析、客户诉求、Demo/原型、共创验证、用户旅程、技术评估结论。
